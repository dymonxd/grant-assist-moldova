<objective>
Create the AI inference module for business idea to company profile conversion, and build all four Server Actions that power the landing page: lookupCompany (IDNO scrape), inferFromIdea (AI inference), saveManualProfile (manual fallback), and savePurchaseNeed (purchase need selection). All actions use the admin Supabase client for anonymous user writes and store the profile ID in the iron-session cookie. Includes Wave 0 unit tests for AI inference and profile actions.

Purpose: These Server Actions are the bridge between the scraping/AI layer (Plan 01) and the UI (Plan 03). They handle data persistence, session management, and the critical anonymous-user-write pattern.

Output: Four working, tested Server Actions callable from Client Components in Plan 03.
</objective>

    1. Create `src/app/actions/profile.ts` with `'use server'` directive and three exported actions:

       **lookupCompany(idno: string)**:
       - Validate IDNO with `validateIdno(idno)`. Return `{ error: string }` if invalid.
       - Call `aggregate(idno)` to scrape all three sources in parallel.
       - Use `createAdminClient()` (NOT the regular server client -- anonymous users have no auth.uid(), RLS would block the insert).
       - Upsert into company_profiles with `{ onConflict: 'idno' }`: set company_name, industry, location, legal_form from merged data, store full raw results in enriched_data JSONB.
       - Get session via `getSession()`, store `data.id` as `session.companyProfileId`, call `session.save()`.
       - Return `{ profile: data, sourceStatus, isPartial, allFailed }`.

       **inferFromIdea(businessIdea: string)**:
       - Validate that businessIdea is non-empty and at least 10 characters. Return `{ error: string }` if too short.
       - Call `inferProfileFromIdea(businessIdea)`.
       - If AI returns null, return `{ error: 'Nu am putut analiza ideea de afacere' }`.
       - Use `createAdminClient()` to insert into company_profiles with: business_idea field set, company_name/industry/location/legal_form from AI output, enriched_data storing `{ source: 'ai_inference', ai_output: result }`.
       - Store profile ID in session cookie.
       - Return `{ profile: data }`.

       **saveManualProfile(fields: { company_name: string, industry: string, location: string, legal_form: string })**:
       - Validate that company_name is non-empty. Return error if missing.
       - Use `createAdminClient()` to insert into company_profiles with provided fields and enriched_data `{ source: 'manual_entry' }`.
       - Store profile ID in session cookie.
       - Return `{ profile: data }`.

       - Use `createAdminClient()` to update the company_profiles row matching `session.companyProfileId`, setting `purchase_need = purchaseNeed`.
       - Return `{ success: true }` on success, `{ error: string }` on failure.
