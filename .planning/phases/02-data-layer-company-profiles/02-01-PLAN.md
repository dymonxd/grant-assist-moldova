    - aggregate() with all three succeeding returns merged data from highest-confidence source, isPartial: false, allFailed: false
    - aggregate() with one failing returns merged from remaining, isPartial: true, allFailed: false

    - aggregate() field merge: if idno.md has company_name but not industry, and srl.md has industry, merged has both
  </behavior>
  <action>
    1. Create `src/lib/sources/idno-md.ts`: Export `scrapeIdnoMd(idno: string): Promise<SourceResult>`. Use AbortController with 8000ms timeout. Fetch `https://idno.md/${idno}` with User-Agent header. Parse HTML with `import * as cheerio from 'cheerio'`. Extract company_name, industry, location, legal_form using CSS selectors (use best-guess selectors like `h1`, `.company-name`, `td:contains("Denumire")` patterns -- these will need live-testing refinement). Return graceful null for any field that doesn't match. On AbortError return status:'timeout', on any other error return status:'error'. Always clearTimeout in finally block.

    2. Create `src/lib/sources/srl-md.ts`: Same pattern as idno-md.ts but fetches `https://srl.md/search?q=${idno}` with confidence 0.8. Different CSS selectors for srl.md's HTML structure. Same timeout and error handling.

This source may primarily provide procurement data, so some profile fields may be null -- that's expected.

    4. Create `src/lib/sources/aggregate.ts`: Export `aggregate(idno: string): Promise<AggregateResult>`. Run all three scrapers with `Promise.allSettled([scrapeIdnoMd(idno), scrapeSrlMd(idno), scrapeOpenMoney(idno)])`. For each settled result, extract the SourceResult (fulfilled) or create an error SourceResult (rejected). Merge fields using confidence-weighted strategy: sort successful results by confidence descending, for each field use first non-null value. Compute `sourceStatus` (per-source status map), `isPartial` (some succeeded AND some failed), `allFailed` (zero successes). Store raw results keyed by source name in `raw` (this goes into enriched_data JSONB).

Test: all succeed (merge correctness), partial failure (isPartial), all fail (allFailed), and cross-source field merging (field from lower-confidence source used when higher-confidence source has null for that field).
